## � Visual Philosophy: 몽글몽글 픽셀 아트

우리는 90년대 PC통신의 따뜻한 향수와 픽셀 아트의 미학을 현대적인 기술로 재해석합니다.

* **Pastel Palette**: 아이보리, 민트, 핑크의 온화한 색감으로 정서적 안정감을 제공합니다.
* **Dynamic Metamorphosis**: 당신의 답변 톤에 따라 나무의 색과 밀도가 실시간으로 변이하는 살아있는 캔버스입니다.

## 🧺 The Gardener’s Protocol: 7단계 인양 로직

AI 정원사 '지미니'는 당신의 기억을 단순히 수집하지 않고, 정성스럽게 인양합니다. [cite: 2026-02-04, 58-61]

1. **The Root (뿌리)**: 유년기, 당신을 처음 설레게 한 보물.
2. **The Sprout (싹)**: 청춘의 주파수를 흔든 노래와 문장.
3. **The Stem (줄기)**: 밤잠을 설치게 했던 첫 번째 몰입의 기억.
4. **The Branch (가지)**: 일상 속에서 당신을 지탱하는 세련된 취향.
5. **The Leaf (잎)**: 누구에게도 말하지 않은 영혼의 안식처.
6. **The Flower (꽃)**: 당신의 모든 취향을 관통하는 하나의 가치.
7. **The Fruit (열매)**: 전 세계 취향 지도(Taste Pin) 위에 맺히는 당신의 고유한 자산.

## 🛠 Technical Stack: 견고한 정원의 뼈대

가장 현대적이고 안정적인 기술을 통해 다정한 경험을 구현합니다.

* **Frontend**: Next.js 14 (App Router), Tailwind CSS
* **AI & Intelligence**: Google Gemini API (7-Stage Logic), `next-intl` (i18n)
* **Infrastructure**: Vercel (Hosting), Supabase (Asset Storage), Firestore (Sync)
* **Graphics**: Custom Pixel Graphics Engine (CSS Step Rounding & Canvas)

## 🚀 Getting Started: 정원으로 들어오는 법

아래 명령어를 통해 로컬 환경에서 직접 정원을 가꿀 수 있습니다.

```bash
# 레포지토리 복제
git clone https://github.com/Lolololotus/taste-tree.git

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

# 🌳 Taste Tree: 나만의 고유함을 키워내는 디지털 정원

> "갓생의 압박에서 벗어나, 오롯이 나의 취향에 침잠하는 시간."

**Taste Tree**는 바쁘게 돌아가는 현대 사회의 효율성 지상주의에서 벗어나, 사용자가 잊고 지냈던 소중한 추억과 고유한 취향을 다시 발견하고 이를 가치 있는 **디지털 자산**으로 키워내는 프로젝트입니다.

## ✨ 핵심 기능 (Key Features)

### 1. 🤖 AI 픽셀 정원사 (AI Pixel Gardener)
- **Google Gemini** 기반의 페르소나 챗봇이 당신의 기억을 안내합니다.
- 90년대 PC 통신 감성의 타자기 효과와 따뜻하고 섬세한 대화를 제공합니다.
- 단순한 대화가 아닌, 사용자의 답변 깊이를 분석하여 **신뢰 점수(Trust Score)**를 산출합니다.

### 2. 🌲 실시간 성장 정원 (Living Garden)
- 대화가 깊어질수록 배경의 나무가 실시간으로 성장합니다.
- **5단계 성장 시스템**: 씨앗(Seed) → 싹(Sprout) → 묘목(Sapling) → 나무(Tree) → 개화(Bloom).
- **Framer Motion**을 활용한 유기적이고 부드러운 픽셀 아트 애니메이션.

### 3. 💎 취향의 자산화 (Taste-to-Earn)
- 당신의 진솔한 이야기는 **Answer_Asset**이라는 데이터 자산이 됩니다.
- 대화 종료 시, Trust Score에 비례하여 **$TASTE 토큰** 보상을(Mock) 획득할 수 있습니다.
- 공식 명칭(Official Name)과 장르를 포함한 구조화된 메타데이터로 저장됩니다.

## 🛠 기술 스택 (Tech Stack)

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS (Pixel Art Aesthetic)
- **Animation**: Framer Motion
- **AI Engine**: Google Gemini API (gemini-1.5-flash)
- **State Management**: React Hooks

## 🚀 시작하기 (Getting Started)

1. **환경 변수 설정**: root 디렉토리에 `.env` 파일을 생성하고 Gemini API 키를 입력하세요.
   ```bash
   GEMINI_API_KEY=your_api_key_here
   ```

2. **의존성 설치**:
   ```bash
   npm install
   ```

3. **개발 서버 실행**:
   ```bash
   npm run dev
   ```
